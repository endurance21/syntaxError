# SyntaxError
A order management application developed in Nodejs using Express

# Getting started

Unzip the downloaded file.

### Installing dependencies:
Enter this command it will install all the dependencies at once:

```
npm install
```
Or you can install them individually:

```
npm install express express-session mysql 
```

### Start the application

```
node app.js
```
### Database

For this application the database  name is 'tray', it contains multiple tables

Go to app.js enter your database username and password, you can use you own database name just make sure it's the same in the app.js file so you can connect to database.
